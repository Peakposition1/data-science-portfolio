# Data Science Portfolio

Welcome to my data science portfolio! This repository contains projects showcasing my skills in **Python, SQL, Data Analysis, Machine Learning, and Time Series Forecasting**.

---

## Projects Overview

### 1. Superstore Sales Analysis
- **Goal:** Analyze a retail dataset to understand sales patterns, product performance, and customer behavior.  
- **Techniques:** Data cleaning, EDA, aggregation, visualization (bar plots, heatmaps, trend analysis).  
- **Key Insights:** Identified top-performing products, regions, and trends; discovered opportunities to optimize sales and inventory.  
- **Portfolio Notebook:** `superstore_sales_analysis.ipynb`  

---

### 2. Online Retail Analysis
- **Goal:** Analyze a global online retail dataset to understand purchase behavior and seasonal trends.  
- **Techniques:** Data preprocessing, handling missing values, transaction analysis, RFM (Recency, Frequency, Monetary) analysis, visualization.  
- **Key Insights:** Recognized key customer segments, peak sales periods, and churn patterns.  
- **Portfolio Notebook:** `online_retail_sales_analysis.ipynb`  

---

### 3. Customer Churn Classification
- **Goal:** Predict which customers are likely to churn using a machine learning classification approach.  
- **Techniques:** Feature engineering, encoding categorical variables, train-test split, model training (Logistic Regression, Random Forest), evaluation (accuracy, confusion matrix).  
- **Key Insights:** Identified important churn drivers and provided actionable recommendations for customer retention.  
- **Portfolio Notebook:** `customer_churn_classification.ipynb`  

---

### 4. Walmart Sales Forecasting (Advanced Time Series)
- **Goal:** Forecast weekly sales for Walmart stores using historical data.  
- **Techniques:** Time series preparation, exploratory analysis, baseline models (Naive & Moving Average), SARIMA modeling, evaluation (MSE & MAE).  
- **Key Insights:** Captured trends and seasonality, predicted holiday spikes, provided actionable recommendations for inventory and staffing.  
- **Portfolio Notebook:** `walmart_sales_forecasting.ipynb`  

---

## Skills Demonstrated

- **Python:** pandas, numpy, matplotlib, seaborn, scikit-learn, statsmodels  
- **SQL:** Data extraction and aggregation  
- **Machine Learning:** Classification, evaluation metrics, feature engineering  
- **Time Series:** Trend analysis, seasonality, SARIMA forecasting  
- **Data Visualization:** Clear plots, dashboards, actionable insights

## How to Run

1. Clone the repository.  
2. Make sure all **CSV files** are in the same folder as their respective notebooks.  
3. Open the notebook in **JupyterLab** or **VS Code**.  
4. Run all cells sequentially to reproduce results.
